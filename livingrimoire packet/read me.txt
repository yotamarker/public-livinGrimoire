this packet is goaled to make it easier to start a new project utilizing
the living grimoire AGI design pattern.

core classes :
base stand alone java classes to utilize the chobit class
which is the living grimoire design pattern. enabling the coder to
add a skill using one line of code

externals : 
classes for hardware and adaptive codes such as
mainactivity and the customised text to speech.
this classes will vary between platforms, programming languages and
different devices.
this also includes the UI xml file.

raw :
voice files for the custom TTS kotlin class in the external folder
see lgkotlin1 raw folder

drawable :

suggested UI pictures : BG pic and icon
